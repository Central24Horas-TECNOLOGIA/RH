# Frontend

## Pastas principais

- `estilos/`: CSS e logos.
- `Exames/`: materiais de apoio do fluxo de prova.
- `fonte/app/`: bootstrap, navegacao e controlador central.
- `fonte/features/`: modulos de tela por dominio.
- `fonte/services/api/`: cliente HTTP por dominio.
- `fonte/shared/`: validacoes e helpers reutilizaveis.
- `fonte/ui/`: layout e componentes compartilhados de interface.

## Estrategia adotada

Frontend mantido em JavaScript modular com ESM, sem migracao parcial para stack hibrida.
