# 09 — Manual do usuário RH

## Objetivo do manual

Este manual ensina o colaborador do RH a usar o Conecta C24h no dia a dia, sem entrar em detalhes técnicos.

## 1. Acessar o sistema

1. Abra o endereço interno do Conecta C24h no navegador.
2. Na tela de login, informe usuário e senha.
3. Clique em **Entrar**.
4. Após autenticar, o sistema abre o **Painel**.

Se a sessão expirar, faça login novamente. Algumas ações sensíveis, como abrir resultado de prova ou retornar ao menu após prova, podem exigir nova autenticação.

## 2. Entender o menu lateral

O menu lateral concentra as áreas principais:

| Menu | Quando usar |
| --- | --- |
| **Painel** | Ver resumo geral e últimos movimentos |
| **E-mails** | Tratar currículos recebidos por e-mail |
| **Histórico** | Consultar provas/testes realizados |
| **Processos** | Criar, editar, acompanhar e encerrar processos seletivos |
| **Candidatos** | Consultar candidatos registrados |
| **Entrevistas** | Criar slots e acompanhar entrevistas |
| **Análise** | Ver relatórios e indicadores |
| **Banco de talentos** | Reaproveitar candidatos para novas vagas |

Atalhos importantes:

- **Novo processo:** cria nova vaga/processo.
- **Nova prova:** inicia fluxo de teste/prova.

## 3. Criar um processo seletivo

1. Acesse **Processos**.
2. Clique em **Novo processo**.
3. Preencha os dados da vaga:
   - vaga;
   - quantidade de vagas;
   - data de encerramento;
   - operação;
   - trilha;
   - nota de corte, se aplicável;
   - textos públicos/observações da vaga, quando houver.
4. Confira as informações.
5. Salve o processo.

Depois de criado, o processo aparece na lista de processos e pode receber candidatos.

## 4. Acompanhar processos

1. Acesse **Processos**.
2. Localize a vaga desejada.
3. Abra os detalhes do processo.

Na tela de detalhes, você pode acompanhar:

- candidatos no processo;
- candidatos aprovados;
- pré-análises de CV;
- entrevistas agendadas;
- ações disponíveis por candidato;
- andamento e status da vaga.

## 5. Tratar currículos recebidos por e-mail

A entrada principal de currículos no fluxo atual é a **caixa de e-mail**.

1. Acesse **E-mails** ou veja o resumo no **Painel**.
2. Localize o e-mail recebido.
3. Use as ações disponíveis:
   - **Abrir detalhes:** ver informações completas do e-mail.
   - **Abrir/baixar CV:** conferir o anexo.
   - **Analisar CV:** o sistema tenta extrair dados e classificar o candidato.
   - **Vincular a processo:** manda o candidato para uma vaga aberta.
   - **Banco de talentos:** guarda o candidato para uso futuro.
   - **Ignorar:** oculta da rotina sem excluir de vez, conforme regra da tela.
   - **Excluir:** remove o e-mail da caixa registrada no sistema.

### Boa prática

Não confie cegamente na classificação automática. Use a análise como apoio. Se o candidato for classificado como **não qualificado**, mas o RH entender que vale seguir, utilize a ação manual disponível.

## 6. Analisar CV

Ao clicar em **Analisar CV**, o sistema tenta identificar:

- nome do candidato;
- e-mail;
- telefone/WhatsApp;
- cidade/bairro;
- experiências;
- competências;
- formação;
- classificação;
- justificativa.

Depois da análise, revise os dados. Se o nome, contato ou classificação vierem incorretos, corrija manualmente quando a tela permitir.

## 7. Vincular candidato a processo

1. Na caixa de e-mail ou banco de talentos, escolha o candidato.
2. Selecione o processo aberto.
3. Confirme a vinculação.
4. O candidato passa a aparecer nos detalhes do processo.

## 8. Usar o banco de talentos

1. Acesse **Banco de talentos**.
2. Use filtros para localizar o candidato.
3. Abra/edite o perfil, se necessário.
4. Se quiser reaproveitar, selecione um processo aberto.
5. Confirme o uso do candidato.

Use o banco de talentos para candidatos bons que não foram aproveitados no momento ou que podem servir para outra vaga.

## 9. Criar slots de entrevista

1. Acesse **Entrevistas**.
2. Crie um novo slot informando:
   - processo;
   - data;
   - hora inicial;
   - hora final;
   - duração;
   - capacidade;
   - observações, se necessário.
3. Salve.

O slot representa uma janela disponível. Se ele não aparecer ao agendar candidato, verifique se:

- o processo está correto;
- a data/hora é futura;
- o slot está ativo;
- ainda há capacidade disponível;
- o candidato está em status que permite entrevista.

## 10. Agendar entrevista para candidato

1. Abra o detalhe do processo.
2. Localize o candidato.
3. Clique na ação de entrevista/agendamento.
4. Escolha um slot disponível.
5. Revise a mensagem/observações.
6. Confirme o agendamento.

Após salvar, o status do candidato deve refletir o agendamento.

## 11. Atualizar status da entrevista

Na tela de entrevistas ou detalhe do processo, atualize o status conforme o caso:

- agendado;
- confirmado;
- reagendado;
- compareceu;
- faltou;
- aprovado;
- eliminado;
- banco de talentos.

## 12. Aplicar nova prova

1. Clique em **Nova prova**.
2. Preencha os dados do candidato e vaga.
3. Escolha nível/trilha conforme o perfil da vaga.
4. Inicie o teste.
5. O candidato responde as etapas.
6. Ao final, o sistema registra a pontuação e salva no histórico.

## 13. Consultar resultado de prova

1. Acesse **Histórico**.
2. Localize o candidato pelo nome, vaga ou data.
3. Abra o resultado.
4. Quando solicitado, informe login/senha de administrador.
5. Confira:
   - nota final;
   - etapas;
   - status;
   - observações;
   - dados para impressão.

## 14. Aprovar candidato

1. No detalhe do processo, localize o candidato.
2. Clique em **Aprovar**.
3. Revise a mensagem padrão.
4. Marque os documentos solicitados.
5. Informe data de comparecimento, se aplicável.
6. Anexe documento se necessário.
7. Confirme.

Após aprovado, o candidato não deve continuar em lista ativa do processo.

## 15. Eliminar candidato

1. No detalhe do processo, localize o candidato.
2. Clique em **Eliminar**.
3. Informe motivo e etapa da eliminação.
4. Confirme.

O histórico deve registrar a decisão.

## 16. Encerrar processo

1. Acesse **Processos**.
2. Abra ou localize o processo.
3. Use a ação de encerramento.
4. Confirme.

Depois de encerrado, o processo deve bloquear novas movimentações internas.

## 17. Emitir relatórios

1. Acesse **Análise**.
2. Escolha relatório de processos ou candidatos.
3. Aplique filtros de período/status, se houver.
4. Visualize na tela ou exporte.

## 18. Erros comuns para o RH

| Situação | O que fazer |
| --- | --- |
| Não consigo entrar | Verificar usuário/senha e se a API está ligada |
| CV não analisa | Conferir se o anexo é PDF/DOCX/DOC/RTF/ODT/TXT válido |
| Candidato não aparece no processo | Confirmar se foi vinculado ao processo correto |
| Slot não aparece | Conferir processo, capacidade, status do slot e status do candidato |
| Resultado pede login | É comportamento esperado de segurança |
| Processo encerrado bloqueia ação | É comportamento correto para evitar movimentação indevida |
